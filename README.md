# Real-Time Sign Language Recognition Using AI

## Project Overview
This project presents a real-time sign language recognition system designed to convert hand gestures into text and speech. The system aims to assist mute individuals in communicating effectively with others.

## Technologies Used
- Python
- TensorFlow
- Convolutional Neural Networks (CNN)
- MobileNetV2
- Random Forest
- Raspberry Pi
- OpenCV

## Features
- Real-time gesture recognition
- Gesture-to-text conversion
- Text-to-speech output
- Support for sign language alphabets
- Lightweight deployment on embedded devices

- Master thesis PDF

- Experimental results
